Lists Usage
Unordered List An unordered list is used to describe items that do not have a specific order. To start this list you use the tag <ul>

Definitions

HTML
HTML stands for Hyper Text Markup Language. It is a nonproprietary language used to build and display web pages.
CSS
CSS stands for Cascading Style Sheet. It is a set of properties and values that describe how a web page should look like.
FTP
FTP stands for File Transfer Protocol. It is used to promote sharing of files, incouurage indirect or implicit use of remote computers, shield a user from variations in file storage systems, and transfer data reliable and efficiently
SSH
SSH stands for Secure Shell. It is a communication standard that provides a secure way to access a computer over an unsecured network.
